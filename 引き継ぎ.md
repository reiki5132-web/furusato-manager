# ふるさと納税 家族管理ツール 引き継ぎドキュメント

作成日：2026年4月21日

---

## 1. アプリの概要

家族（りの・母）のふるさと納税記録を一元管理するスマホ対応Webアプリ。

- **URL**：GitHub Pages で公開（`/furusato-manager/` パス）
- **データ保存**：ブラウザの localStorage（サーバー不要・外部送信なし）
- **対応デバイス**：スマートフォン優先設計。PCでも動作する
- **PWA対応**：スマホのホーム画面に追加してアプリのように使える

## 2. 画面・機能一覧

### 🏠 ホーム（ダッシュボード）
- りの・母それぞれの寄付状況カードを表示
- 年間控除上限額・寄付済み金額・残り枠を表示
- 使用率に応じて色が変わる（緑 → 黄 → 赤）
- ふるさと納税の締め切り（12/31）まで60日を切ると警告バナーが出る
- 画面右上のセレクトで年度を切り替え（2024/2025/2026）

### 📋 寄付記録
- 全寄付の一覧（新しい順）
- ユーザーでフィルタリング可能
- 新規追加ボタンでフォームが開く
- 各記録をタップすると編集・削除できる

**入力項目：**

| 項目 | 内容 |
|------|------|
| 誰の寄付 | りの / 母 |
| 寄付日 | 日付（年度自動判定） |
| 自治体名 | テキスト入力 |
| 寄付金額 | 数値入力 |
| 返礼品名 | テキスト入力 |
| カテゴリ | 食品・飲料 / 日用品 / 家電 / 工芸品 / アウトドア / その他 |
| 税申告方法 | ワンストップ特例 / 確定申告 |
| 受領証の受取 | チェックボックス |
| 受取予定時期 | テキスト（「3月ごろ」など自由入力） |

### 🎁 返礼品トラッカー
- 返礼品のある寄付記録を一覧表示
- 配送ステータスを3段階で管理：**未発送 → 発送済み → 受取済み**
- ボタンを押すだけでステータスが進む
- 画面上部にステータス別の件数サマリーを表示

### ⚙️ 設定
- りの・母それぞれの**年間控除上限額**を登録
- 上限額がわからない場合は**簡易計算機**で目安を算出
  - 年収（10段階）と家族構成（独身 / 夫婦 / 子1人 / 子2人）を選ぶと自動計算
  - ただし目安値なので、正確な金額はふるさと納税サイトの計算ツールで確認推奨

## 4. データ構造

ブラウザの localStorage に JSON 形式で保存されます。

**保存キー：** `furusato-manager-data`

### AppData（アプリ全体のデータ）

```
{
  users: {
    rino: { id: "rino", name: "りの", limitAmount: 120000 },
    haha: { id: "haha", name: "母",   limitAmount: 80000  }
  },
  donations: [ ...寄付記録の配列... ],
  activeUser: "rino"
}
```

### Donation（寄付記録1件）

```
{
  id: "uuid",                    // 自動生成ID
  userId: "rino",                // 誰の寄付
  date: "2025-11-01",            // 寄付日
  year: 2025,                    // 年度
  municipalityName: "北海道白糠町",
  amount: 10000,                 // 寄付金額（円）
  returnGiftName: "いくら醤油漬け",
  returnGiftCategory: "食品・飲料",
  taxMethod: "onestop",          // "onestop" or "kakutei"
  receiptReceived: true,         // 受領証受取済み
  giftStatus: "received",        // "unsent" / "shipped" / "received"
  giftScheduledMonth: "3月ごろ"
}
```

## 9. データのバックアップ・移行

データはブラウザの localStorage に保存されているため、**ブラウザや端末が変わるとデータが引き継がれません。**

### バックアップ方法（手動）

ブラウザの開発者ツール（F12）→ Application → localStorage → `furusato-manager-data` の値をコピーしてテキストファイルに保存。

### 別端末への移行

1. 旧端末でバックアップのJSON文字列をコピー
2. 新端末のブラウザで同じURLを開く
3. 開発者ツール → Console で以下を実行：
   ```javascript
   localStorage.setItem('furusato-manager-data', 'コピーしたJSON文字列');
   ```
4. ページをリロード
